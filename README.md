# Spam Email Detection using Machine Learning

## Problem Statement
Detect whether an email/message is spam or not using Machine Learning.

## Dataset
Custom CSV dataset with labeled spam and ham messages.

## Technologies Used
- Python
- Google Colab
- Pandas, NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Classifier

## Model
Multinomial Naive Bayes

## Accuracy
Achieved good accuracy on test data.

## How to Run
1. Open the notebook in Google Colab
2. Run all cells step by step
3. Test with your own message
